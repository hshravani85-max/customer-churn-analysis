## 📌 Problem Statement
Customer churn is a major challenge for businesses as it directly impacts revenue.  
This project analyzes customer data to identify key factors that lead to churn and provides actionable strategies to improve retention.

## 🚀 Project Overview
This project focuses on analyzing customer behavior, identifying churn patterns, and building a dashboard to support data-driven decision-making.

## 🎯 Objectives
- Identify key factors influencing customer churn  
- Analyze customer behavior and usage patterns  
- Detect high-risk customer segments  
- Provide business recommendations to reduce churn  

## 🛠️ Tools & Technologies
- 🐍 Python (Pandas, NumPy)  
- 🗄️ SQL  
- 📊 Power BI  
- 📈 Excel  

## 📊 Key Metrics
- 👥 Total Customers: **7032+**  
- 📉 Churn Rate: **26.58%**  

## 📊 Key Insights
- 🔴 Around **1 in 4 customers churn**, indicating retention issues  
- 📉 **Month-to-month contracts** have highest churn  
- ⚠️ **New customers (low tenure)** are most likely to churn  
- 💰 Higher **monthly charges** increase churn probability  

## 💡 Business Recommendations
- ✅ Encourage long-term subscription plans  
- 🚀 Improve onboarding experience for new customers  
- 🎯 Offer personalized pricing and discounts  
- 📊 Focus on high-risk customer segments
